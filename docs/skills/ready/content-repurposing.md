# Content Repurposing Skill

Takes one piece of source content and transforms it into ready-to-publish
formats for different platforms. Works for any niche or audience.

---

## Workflow

1. **Receive source content** — text, transcript, idea, URL, or voice note summary
2. **Analyze the content** — extract: core idea, tone, audience, niche
3. **Suggest formats** — based on the niche, propose 3–5 relevant output formats
   and ask the user which ones to produce (or produce all if user says "всё")
4. **Produce each format** — tailored to platform conventions
5. **Output as cards** — one card per format, with copy button
6. **Offer iteration** — tone adjustment, length, CTA change

---

## Step 2: Content Analysis

Before writing anything, extract:
- **Core idea** — one sentence: what is this content actually about?
- **Tone** — educational / inspirational / entertaining / personal / promotional
- **Niche** — what field/industry does this belong to?
- **Audience** — who is this for? (beginners / professionals / buyers / followers)
- **Key hook** — the most attention-grabbing element

Show this analysis briefly before proposing formats, so the user can correct if needed.

---

## Step 3: Format Selection

Based on niche and tone, suggest the most relevant formats from this list.
Do not offer all formats every time — pick 3–5 that make sense for the content.

| Format | Best for | Length |
|--------|----------|--------|
| **Telegram post** | Communities, expert channels | 150–400 words |
| **Instagram / Reels caption** | Visual content, lifestyle, product | 50–150 words + hashtags |
| **Short-form hook** | First 3 seconds of video, opening line | 1–2 sentences |
| **Email newsletter** | Nurture sequences, announcements | 200–500 words |
| **YouTube / video description** | SEO, context for video | 100–200 words |
| **Story sequence** | Instagram/Telegram Stories, 3–5 slides | 20–40 words per slide |
| **Headline variants** | A/B testing, ads, article titles | 3–5 options |
| **Thread / carousel** | Twitter/X, LinkedIn, Telegram | 5–8 points |
| **CTA block** | Any format, at the end | 1–3 sentences |

---

## Format Rules

### Telegram post
- Open with a strong first line (the hook) — no preamble
- Use short paragraphs (2–4 lines max)
- One clear idea per post
- End with either a question, CTA, or a punchy conclusion
- Emoji: use sparingly, only if it fits the tone (0–3 per post)
- No hashtags (Telegram search doesn't benefit from them)

### Instagram / Reels caption
- First line = hook (must work without "more" expansion)
- Keep it scannable — line breaks every 1–2 sentences
- End with a question or CTA to drive comments
- 3–10 hashtags at the end, niche-relevant
- Tone: warmer and more personal than Telegram

### Short-form hook
- Must create a pattern interrupt in under 3 seconds
- Formats that work: question / bold claim / unexpected fact / "I did X and here's what happened"
- No filler words, no "Hi everyone", no intro
- Write 3 variants — let the user choose

### Email newsletter
- Subject line: specific, curiosity-driven, no clickbait
- Opening: connect to a pain point or moment
- Body: one idea, developed with story or data
- CTA: one action only — clear and low-friction
- Sign-off: personal, not corporate

### Story sequence (3–5 slides)
- Slide 1: Hook / tension
- Slide 2–3: Context / proof / story
- Slide 4: Resolution or insight
- Slide 5: CTA or question
- Each slide: max 40 words, designed to be read in 3 seconds

### Headline variants
- Write 5 options in different styles: question / number / how-to / bold claim / emotional
- Mark which style each is
- Keep under 10 words each

### Thread / carousel
- Point 1 = hook (standalone, shareable)
- Points 2–7 = one idea each, short and punchy
- Last point = conclusion + CTA
- Each point: 1–3 sentences max

---

## Tone Adaptation

Always match the platform's native tone:

| Platform | Tone |
|----------|------|
| Telegram | Direct, expert, slightly informal |
| Instagram | Warm, visual, personal |
| Email | Conversational but structured |
| YouTube | Informative, search-optimized |
| LinkedIn | Professional, insight-driven |

If the user has a defined personal voice — ask them to describe it or paste an example,
then match it across all formats.

---

## Output Format

One card per format. Each card contains:
- Format name + platform icon
- Ready-to-publish text
- Copy button
- Character / word count

After all cards — a short distribution note:
which format to post first, which to schedule, which to A/B test.

## Example

**Source:** "Снял ролик за 2 часа с помощью ИИ и продал его клиенту за $500"

**Analysis:**
- Core idea: AI video production = fast income
- Tone: personal, inspirational, proof-based
- Niche: AI creators, freelancers
- Hook: the result ($500, 2 hours)

**Suggested formats:** Telegram post · Reels caption · Short-form hook × 3 · Story sequence

---

**Telegram post:**
```
Снял ролик за 2 часа и продал его за $500.

Не монтировал вручную. Не нанимал команду.
Весь пайплайн — ИИ от промпта до финального файла.

Клиент доволен. Я доволен. Инструменты окупились за один проект.

Вот как это работает: [ссылка / продолжение]
```

**Reels caption:**
```
2 часа → готовый ролик → $500 в кармане 🤝

Больше не трачу дни на производство.
ИИ делает тяжёлую работу, я — креативные решения.

Хочешь узнать какой пайплайн использую? Пиши в комменты 👇

#aicontentcreator #aigenerator #freelance #contentcreation
```

**Hook variants:**
```
1. "Что если ролик для клиента можно сделать за 2 часа?"
2. "Я продал ИИ-видео за $500. Вот весь процесс."
3. "Большинство криэйторов тратят дни на то, что ИИ делает за час."
```
