# Offer Builder Skill

Turns a raw service description or idea into a structured, ready-to-use offer.
Works for freelancers, consultants, course creators, agencies, and entrepreneurs.
Output is a complete offer card: name, positioning, pain, solution, result, price, CTA.

---

## Workflow

1. **Receive input** — what the user does, sells, or wants to package
2. **Ask clarifying questions** if needed (see below) — max 3–4 questions
3. **Build the offer** — fill all 8 components
4. **Output offer card** — structured, copy-ready, with copy buttons
5. **Suggest variants** — offer a stronger name or CTA alternative if relevant

---

## Clarifying Questions (ask only what's missing)

If the user gives only a rough idea, ask:

1. **Кто твой клиент?** — кто конкретно это покупает, какая у него боль
2. **Какой результат** клиент получает после работы с тобой?
3. **Формат:** это разовая услуга, пакет, курс, подписка?
4. **Цена:** уже есть или нужно помочь сформулировать?

Never ask all 4 at once — only what's genuinely missing from the input.

## Offer Structure (8 components)

### 1. Название (Name)
Короткое, конкретное, запоминающееся.
- Должно отражать результат, а не процесс
- Не "консультация по маркетингу" → "Стратегия роста за 1 день"
- Формулы: [Результат] за [срок] / [Действие] + [для кого] / [Название-бренд]

### 2. Для кого (Target)
Одна конкретная аудитория.
- Не "для всех предпринимателей"
- Да: "для фрилансеров без стабильного потока клиентов"

### 3. Боль (Pain)
Проблема которую решает оффер. Формулируй языком клиента — не экспертным.
- Что его раздражает / пугает / тормозит прямо сейчас?
- Какова цена бездействия?

### 4. Решение (Solution)
Что конкретно ты делаешь. Без воды.
- Формат работы
- Этапы или включения
- Чем отличается от того что клиент мог бы сделать сам

### 5. Результат (Result)
Что клиент получает на выходе. Конкретно.
- Осязаемый результат: файл, стратегия, видео, продажи, время
- Эмоциональный результат: уверенность, ясность, снятие стресса
- Если возможно — цифры: "за 3 дня", "вместо 2 недель"

### 6. Формат и сроки (Format)
- Как проходит работа: онлайн / оффлайн / асинхронно
- Длительность: разово / неделя / месяц
- Количество итераций / встреч / правок

### 7. Цена и условия (Price)
- Сумма или диапазон
- Что входит
- Что НЕ входит (важно для управления ожиданиями)
- Варианты оплаты если есть

### 8. CTA (призыв к действию)
Одно конкретное действие. Низкий порог входа.
- Не "купи сейчас" — слишком давит
- Да: "напиши мне и я расскажу подходит ли это тебе"
- Да: "запишись на бесплатный разбор"
- Да: "пиши [кодовое слово] в комментарии"

## Усилители оффера

После базового оффера — проверь и добавь если уместно:

**Социальное доказательство:**
- Есть ли кейсы / результаты клиентов? Добавь одну строку.

**Ограничение:**
- Количество мест / дедлайн / условие — создаёт реальную причину действовать сейчас
- Только если честное, не искусственное

**Гарантия:**
- Снижает риск для клиента
- "Если после первой сессии не увидишь ценности — верну деньги"

**Антивозражение:**
- Самое частое "но..." клиента — закрой его прямо в оффере

---

## Output Format

Единая карточка оффера с разделами и кнопками копирования.
Дополнительно — короткая версия оффера (2–3 предложения) для поста или питча.

---

## Tone Rules

- Писать языком клиента, не эксперта
- Конкретика > абстракции
- Результат > процесс
- Никаких "уникальных авторских методологий" без расшифровки
- Никакого "комплексного подхода" — что это значит конкретно?

## Пример

**Вход:** "Я делаю AI-видео для брендов, хочу упаковать это как услугу"

**Название:** AI-видео за 5 дней

**Для кого:** Для брендов и онлайн-школ которым нужен контент но нет времени на долгое производство

**Боль:** Производство видео занимает недели, стоит дорого и требует команды. А контент нужен уже сейчас.

**Решение:** Я создаю готовый видеоролик с помощью AI-пайплайна — от концепта до финального файла — без съёмочной группы и долгих согласований.

**Результат:** Готовый ролик через 5 рабочих дней. Форматы для Reels, YouTube и Stories. Без правок в стиле "переделай всё".

**Формат:** Онлайн, асинхронно. Бриф → концепт на согласование → финальный файл. Включает 2 раунда правок.

**Цена:** от $300 за ролик. Пакет 3 ролика — $750.

**CTA:** Напиши мне описание своего продукта — скажу подойдёт ли формат и покажу примеры похожих работ.
