# Antigravity + n8n: Step-by-Step Guide (Описание)

## Основные концепции
- **Vibe Coding**: Разработка через промпты, где AI берет на себя всю техническую реализацию.
- **Связка n8n + Antigravity**: 
    - n8n выступает как "мозг" и бэкенд (логика, интеграции с API, базы данных).
    - Antigravity выступает как фронтенд и интерфейс (генерация UI, форм, дэшбордов).
- **MCP Server в n8n**: Секрет подключения — использование MCP сервера внутри n8n для того, чтобы Antigravity мог "видеть" и запускать ворклоу.

## Шаги реализации (из видео)
1. Настройка MCP сервера в n8n.
2. Генерация JSON-конфигурации для Antigravity.
3. Промпт №1: Создание логики (подключение к n8n ворклоу).
4. Промпт №2: Дизайн и интерфейс оболочки.
5. Выгрузка готового проекта на свой хостинг.

## Преимущества перед Lovable
- Полная свобода хостинга (Antigravity позволяет выгружать код).
- Глубокая интеграция с локальными инструментами.
