# AI-автоматизація (3).pdf - Содержание курса

Практичний онлайн-курс від діючого AI-фахівця з автоматизації в США "AI Integrator".

## Структура курса
- Модуль 0: Переднавчання. Хто такий AI-автоматизатор.
- Модуль 1: AI-моделі (ChatGPT, Claude, Gemini). Промпт-інжиніринг.
- Модуль 2: Make, n8n, Zapier - основи автоматизації.
- Модуль 3: Бізнес-процеси та цілі.
- Модуль 4: Маркетинг департамент. ClickFunnels, ActiveCampaign.
- Модуль 5: Sales департамент. Pipedrive, GoHighLevel.
- Модуль 6: Support департамент. Intercom.
- Модуль 7: Інші бізнес-юніти (HR, Фінанси).
- Модуль 8: Монетизація навичок. Upwork, LinkedIn.
- Модуль 9: Масштабування та юридичні нюанси.

## Ключевые инструменты
- n8n ( REST API, Webhooks, Бази даних, Telegram AI Агенти)
- LightRAG (упоминается как концепция "Мозг для агента")
- Ollama / Gemma (локальные модели)
- Make.com / Zapier
