# LightRAG: Я подключил своим агентам МОЗГ (Транскрипт)

## Основные тезисы
- **Проблема**: Агенты забывают контекст, сессии сбрасываются (например, в 4 утра у OpenAI), контекст раздувается (50k+ токенов на системные промпты и память).
- **Решение**: LightRAG — графовая база знаний (Knowledge Graph), доступная через MCP.
- **Преимущества**:
    - Память живет вне контекста сессии.
    - Инкрементальное обновление (не нужно перестраивать весь граф при добавлении документа).
    - Дедупликация сущностей (одна нода на понятие, даже если оно в 10 документах).
    - Работает через MCP (один конфиг для всех агентов: Claude Code, OpenClaw, Cursor).

## Технический стек (из видео)
- LightRAG Engine.
- Neo4j (Graph DB) / Qdrant (Vector DB).
- Модели: Gemma 4 (local) или Gemini Flash (API).
- MCP протокол.
