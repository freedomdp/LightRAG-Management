# Интеграция BrowserAct: Руководство для n8n, Make и Zapier

Этот документ объясняет, как запускать ворклоу BrowserAct и передавать данные напрямую в ваши платформы автоматизации.

## 1. Основные методы интеграции
Существует два основных способа подключения BrowserAct к n8n/Make:
- **Прямой HTTP API**: Лучше всего подходит для n8n. Запустите конкретную задачу BrowserAct и дождитесь ответа.
- **Интеграция через Webhooks**: Рекомендуется для длительных процессов. BrowserAct отправляет POST-запрос на ваш URL-адрес Webhook в n8n после завершения извлечения данных.

## 2. Интеграция с n8n
Чтобы запустить задачу BrowserAct из n8n:
1. **HTTP Request Node**:
   - **Метод**: `POST`
   - **URL**: `https://api.browseract.com/v1/tasks` (проверьте конкретный эндпоинт в вашей панели управления)
   - **Заголовки**: `Authorization: Bearer ВАШ_API_KEY`
   - **Тело (Body)**: Укажите ID вашего ворклоу и любые динамические параметры (например, поисковый запрос из сообщения в Telegram).
2. **Получение данных**:
   - Создайте узел **n8n Webhook Node**.
   - Скопируйте URL вебхука и вставьте его в поле «Webhook URL» в настройках вашего ворклоу в BrowserAct.
   - BrowserAct автоматически отправит структурированный JSON-результат (например, бизнес-лиды, транскрипты YouTube) обратно в n8n.

## 3. Интеграция с Make (Integromat)
- Используйте модуль **BrowserAct** в Make (если доступен) или универсальный модуль **HTTP Make an API Call**.
- Логика остается прежней: Запуск по POST -> Получение через Webhook.

## 4. Стратегические паттерны автоматизации
> [!TIP]
> **Паттерн: Автоматизатор «Золотого лида»**
> 1. n8n (Триггер: Новая строка в Google-таблицах)
> 2. BrowserAct (Действие: Поиск в Google Maps соцсетей и имейлов)
> 3. n8n (Действие: Фильтрация лидов с валидными профилями LinkedIn)
> 4. n8n (Действие: Добавление в кампанию охвата Instantly)

## 5. Безопасность и лимиты
- Всегда храните свой API-ключ BrowserAct в разделе **Credentials** (Учетные данные) n8n/Make или в переменных окружения.
- Следите за лимитом одновременных задач (Task Concurrent Limit), чтобы избежать ошибок 429 ворклоу.
