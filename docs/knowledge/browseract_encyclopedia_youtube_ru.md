# Энциклопедия BrowserAct: Аналитика и автоматизация YouTube

Этот документ описывает полный набор инструментов автоматизации YouTube от BrowserAct, предназначенных для работы **Интеллектуального ядра (Intelligence Engine)** стратегии M.O.N.E.Y.

## 1. Обзор модулей
- **youtube-search-api-skill**: Поиск самых популярных видео по ключевым словам, количеству просмотров или дате загрузки.
- **youtube-channel-api-skill**: Массовое извлечение метаданных каналов (подписчики, общее количество видео, ссылки).
- **youtube-influencer-finder-api-skill**: Поиск ключевых авторов в конкретных нишах для охвата (outreach).
- **youtube-video-api-skill**: Глубокое извлечение метаданных конкретных видео (лайки, описания, теги).
- **youtube-batch-transcript-extractor-api-skill**: Массовое извлечение транскриптов (текста) из плейлистов или результатов поиска.
- **youtube-transcript-analysis-api-skill**: Ядро интеллектуального движка — продвинутый ИИ-анализ содержания видео.
- **youtube-comments-api-skill**: Сбор и анализ настроений аудитории и вопросов пользователей.

## 2. Методология анализа по 8 измерениям (8-Dimension Analysis Framework)
При анализе транскриптов с помощью ИИ BrowserAct система использует методологию из 8 этапов:

### I. Структура контента и «Крючок» (Hook)
Анализ архитектуры повествования. Какой открывающий крючок используется? Приведите цитату. Определите структуру (например, PAS, Путь Героя).

### II. Ключевое сообщение
Извлечение «Единой основной точки зрения» и подтверждающих аргументов.

### III. Персонаж аудитории и намерение (Intent)
Кто смотрит видео (новичок против эксперта) и зачем (намерение: обойти проблему, научиться, купить, подтвердить догадку).

### IV. Болевые точки и эмоциональная кривая
Картирование явных и скрытых проблем, которые затрагивает автор. Отслеживание изменения эмоционального состояния зрителя.

### V. Драйверы вовлеченности
Определение факторов, способствующих репостам, «ага-моментов» и запоминающихся концепций.

### VI. Доказательства и доверие
Оценка сигналов авторитетности, ссылок на экспертов и социальных доказательств в видео.

### VII. Бизнес-модель и конверсия
Деконструкция стратегии монетизации (доход от рекламы, курсы, партнерки) и призывов к действию (CTA).

### VIII. Категории контентных пробелов
Стратегическое выявление «белых пятен»:
- **Слабости автора**: Логические ошибки.
- **Нерешенные вопросы зрителей**: Чего всё еще не хватает?
- **Ниши индустрии**: Ракурсы, которые никто не освещает.

## 3. Продвинутые возможности
### A. Массовое извлечение транскриптов
- Оптимизировано для пакетной обработки больших наборов видео.
- Автоматически обрабатывает субтитры на разных языках.
- Прямой вывод в удобном для ИИ формате Markdown.

### B. Интеллект в комментариях
- Выявляет повторяющиеся вопросы в разделе комментариев.
- Определяет «вайб» (настроение) сообщества вокруг канала.
- Идеально подходит для поиска болевых точек клиентов перед первым контактом.

## 4. Рекомендации по использованию
- **Управление скоростью**: При извлечении транскриптов рекомендуется обрабатывать видео группами по 10-20 штук, чтобы избежать таймаутов сессии.
- **Работа с промптами**: Используйте **youtube-transcript-analysis-api-skill** именно тогда, когда вам нужны стратегические инсайты, а не просто сырой текст.
