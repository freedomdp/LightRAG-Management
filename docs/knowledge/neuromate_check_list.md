# Чек-лист проверки на галлюцинации нейросети (Neuromate)

## Пошаговое руководство
1. **Общая оценка**: Поиск "красных флагов" (числа, даты, имена, ссылки).
2. **Fact-checking**:
    - Перекрестная проверка в 2-3 источниках.
    - Проверка существования ссылок (нейросети часто их выдумывают).
    - Валидация цитат через поисковики.
3. **Логика и связность**: Проверка на отсутствие внутренних противоречий.
4. **Соответствие промту**: Не "уплыл" ли агент от задания.
5. **Плагиат**: Проверка на компиляцию без новой ценности.

## Методы борьбы (Guardrails)
- System prompt с жесткими границами ответственности.
- Явный выход на "не знаю" вместо выдумки.
- Проверка региональной специфики (законы РФ).
