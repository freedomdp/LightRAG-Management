# Кейсы BrowserAct: Фреймворк исполнения M.O.N.E.Y.

Этот документ содержит пошаговые сценарии объединения навыков BrowserAct для достижения высокого возврата инвестиций (ROI) в ИИ-агентстве.

## Кейс 1: Движок поиска «болевых точек» локального бизнеса
**Стратегия**: Картирование лидов + Семантический анализ отзывов.
1. **Обнаружение**: Используйте **google-maps-search-api-skill**, чтобы найти «Стоматологические клиники в Берлин» с рейтингом < 4.0.
2. **Анализ**: Извлеките текст недавних негативных отзывов с помощью **google-maps-api-skill**.
3. **Интеллект**: Передайте текст отзывов в **LightRAG** для извлечения повторяющихся жалоб (например, «Долгое время ожидания», «Грубый персонал»).
4. **Охват (Outreach)**: Отправьте персонализированное письмо через Instantly: *«Я заметил, что пациенты жалуются на время ожидания. Я могу внедрить ИИ-голосового агента для автоматизации записи и сокращения очередей».*

## Кейс 2: YouTube-архитектор конкурентной борьбы
**Стратегия**: Анализ контента конкурентов для создания лучших продуктов.
1. **Таргетинг**: Найдите 10 самых популярных видео в нише «Ноотропы» с помощью **youtube-search-api-skill**.
2. **Синтез**: Запустите **youtube-transcript-analysis-api-skill**, чтобы получить отчет по 8 измерениям для каждого видео.
3. **Выявление пробелов**: Определите «Ниши индустрии» (Измерение VIII) — темы, которые упустили конкуренты.
4. **Исполнение**: Создайте лендинг или видео, решающее именно эти пробелы, захватывая аудиторию, которая осталась без ответов.

## Кейс 3: Автоматизированный мониторинг фриланс-бирж (The Hustler)
**Стратегия**: Аудит высокобюджетных заказов в режиме реального времени.
1. **Мониторинг**: Используйте **Web Search Scraper** для отслеживания новых заказов по теме «AI Implementation» на Upwork/Fiverr.
2. **Квалификация**: **Web Research Assistant** посещает профиль клиента в LinkedIn (найденный через Scraper) для проверки их бюджета и размера компании.
3. **Пре-ответ**: ИИ генерирует ответ на основе описания вакансии, уже цитируя аналогичные кейсы, найденные в вашей базе знаний **LightRAG**.

## Кейс 4: Движок личного бренда (Влияние каналов)
**Стратегия**: Поиск инфлюенсеров для партнерства.
1. **Обнаружение**: Используйте **youtube-influencer-finder-api-skill**, чтобы найти каналы с >100 тыс. подписчиков, посвященные «Low Code Automation».
2. **Проверка**: Проверьте вовлеченность их недавней аудитории с помощью **youtube-comments-api-skill**.
3. **Обогащение**: Получите их контактные адреса электронной почты через **business-contact-social-links-skill**.
4. **Результат**: Верифицированный список партнерских контактов для массового охвата.
