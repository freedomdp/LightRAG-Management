# Энциклопедия BrowserAct: Автоматизация Google Maps

Данный документ представляет собой технический обзор модулей BrowserAct для извлечения данных из Google Maps, необходимых для работы **Поискового движка (Mapping Engine)** в рамках стратегии M.O.N.E.Y.

## 1. Основные модули
- **google-maps-search-api-skill**: Оптимизирован для массового поиска локальных компаний по ключевым словам и координатам/названиям локаций.
- **google-maps-api-skill**: Специализирован на извлечении подробных данных о конкретных местах в Google Maps, включая отзывы и полные метаданные.

## 2. Ключевые возможности
### A. Интеллектуальный поиск
- Поддержка запросов на естественном языке (например, «Медицинские центры в Дубае»).
- Точный поиск на основе координат для создания карт высокой плотности.
- **Многорегиональное извлечение**: Возможность обхода нескольких районов или городов за одну задачу.

### B. Глубина данных
Для каждого найденного лида BrowserAct извлекает:
- **Идентификация бизнеса**: Название, категория, статус верификации.
- **Контактные данные**: Номер телефона, URL веб-сайта (база для навыка Business Contact).
- **Социальное подтверждение**: Рейтинг (0-5), общее количество отзывов и фрагменты последних отзывов.
- **Операционные данные**: Полный адрес, Google Maps CID/Place ID, часы работы.

### C. Обход защиты (Anti-Bot)
- **Автоматическое решение капчи**: Встроенная обработка проверок безопасности Google.
- **Скрытный браузинг**: Имитация поведения человека для предотвращения блокировки IP при массовом скрапинге.
- **Динамические User-Agents**: Автоматическая ротация профилей браузера для каждой сессии.

## 3. Интеграция со стратегией M.O.N.E.Y.
> [!TIP]
> Используйте **google-maps-search-api-skill** для создания первоначального списка в базе `agency_leads.db`. Затем эти лиды должны быть переданы в **business-contact-social-links-skill** для обогащения данными.

## 4. Технические ограничения
- Минимальный радиус поиска: 500 м (рекомендуется для картографирования городов).
- Лимиты запросов: Зависят от тарифного плана подписки BrowserAct.

## 5. Паттерны использования (Промпты)
Для достижения наилучших результатов используйте структурированные промпты:
> «Найди все "агентства по автоматизации ИИ" в Лондоне в радиусе 5 км от центра города. Извлеки названия, рейтинги и веб-сайты. Решай все капчи автоматически».
