# 4 техники остановки галлюцинаций ИИ-агентов

## 1. Graph-RAG
В отличие от векторного поиска, Graph-RAG сохраняет структурные связи (Knowledge Graph). Это позволяет агенту выполнять точные расчеты (AVG, SUM) и предоставлять верифицируемые факты вместо поиска похожих кусков текста.

## 2. Семантический выбор инструментов (Semantic Tool Selection)
Фильтрация инструментов через векторное сходство. Агенту отправляются только 3–5 самых релевантных инструментов вместо десятков, что снижает ошибки на 86%.

## 3. Нейросимволические барьеры (Neurosymbolic Guardrails)
Жесткое кодирование правил на уровне фреймворка. Параметры вызова инструмента проверяются Python-функцией *перед* выполнением. Если правило нарушено, вызов блокируется.

## 4. Multi-Agent Validation (Validator-Critic)
- **Executor**: Выполняет задачу.
- **Validator**: Проверяет вывод на ошибки.
- **Critic**: Утверждает результат или отправляет на доработку.
Командный подход эффективнее одиночного агента.
