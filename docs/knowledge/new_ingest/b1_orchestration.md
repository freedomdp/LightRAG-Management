# ИИ-агенты в компании: автономные помощники (B1/EY)

## Отличия: Автоматизация vs RPA vs ИИ-агенты
- **Автоматизация**: Жесткие инструкции, отсутствие обучения.
- **RPA**: Имитация действий человека (перенос данных), работа по четким правилам. Нет понимания контекста.
- **ИИ-агенты**: Эволюция автоматизации. Имитация мышления, работа с неструктурированными данными, адаптивность к изменениям.

## Области применения
- **Персонализация**: Анализ поведения клиентов в ритейле.
- **Логистика**: Динамическое планирование маршрутов.
- **Безопасность**: Антифрод в реальном времени.

## Orchestration Layer
Критическая концепция для крупного бизнеса — единый центр управления множеством агентов для обеспечения безопасности и соблюдения стандартов.
