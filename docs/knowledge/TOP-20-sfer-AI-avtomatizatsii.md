## МАЛИЙ БІЗНЕС (1-20 людей)
Тут AI найшвидше окупається, бо замінює ручну роботу власника й команди.

### 1. Онлайн-магазини (eCommerce)
**Що автоматизуємо:** автозамовлення, чеки, листи, аналітика продажів.
**Інструменти:** Shopify + Make + ChatGPT + Google Sheets.
**Ефект:** економія 10 год/тиждень, +6–12% до виручки.

### 2. Beauty-індустрія (салони, клініки, фітнес)
**Що автоматизуємо:** нагадування клієнтам, CRM, розклади, оплати.
**Інструменти:** Notion CRM + Telegram-бот + Google Calendar.

### 3. Маркетинг-агенції / таргетологи
**Що автоматизуємо:** автоматичні звіти, AI-копірайтинг, онбординг клієнтів.
**Інструменти:** ChatGPT + Make + Airtable + Meta Ads API.

### 4. Освітні продукти, коучі, експерти
**Що автоматизуємо:** CRM, розсилки, AI-контент, контроль прогресу студентів.
**Інструменти:** ChatGPT + Notion + Zapier + Stripe.

### 5. Сфера послуг (ремонт, клінінг, дизайн)
**Що автоматизуємо:** заявки з сайту, нагадування, рахунки, фідбек.
**Інструменти:** Google Forms + Make + Gmail + Sheets.

### 6. Кав’ярні, ресторани, доставка
**Що автоматизуємо:** бронювання, акції, відгуки, програми лояльності.
**Інструменти:** Telegram-бот + Make + ChatGPT + Sheets.

### 7. Блогери / контент-креатори
**Що автоматизуємо:** сценарії, AI-редактура текстів, контент-план.
**Інструменти:** ChatGPT + Notion AI + Descript.

### 8. Нішеві онлайн-курси / мікроакадемії
**Що автоматизуємо:** AI-викладачі, бот-куратори, автоматичні звіти.
**Інструменти:** ChatGPT + Typeform + Make.

### 9. Фрілансери (SMM, дизайн, копірайт)
**Що автоматизуємо:** AI-асистент для клієнтів, шаблони, нагадування.
**Інструменти:** ChatGPT + Notion + Zapier.

### 10. Нерухомість / агенти з продажу
**Що автоматизуємо:** автоматичні листи клієнтам, CRM-нагадування, AI-аналіз угод.
**Інструменти:** HubSpot + Make + ChatGPT + Google Sheets.

## СЕРЕДНІЙ БІЗНЕС (20-100 людей)
AI тут вирівнює процеси, підвищує продуктивність відділів і прибуток.

### 11. B2B-агентства та сервісні компанії
**Що автоматизуємо:** AI-обробка лідів, CRM-потоки, email-нагадування.
**Інструменти:** HubSpot + ChatGPT API + Make.

### 12. HR / рекрутинг-агенції
**Що автоматизуємо:** AI-аналіз резюме, класифікація кандидатів, автоматизація онбордингу.
**Інструменти:** ChatGPT + Notion + Google Drive + Make.

### 13. Будівельні компанії / девелопери
**Що автоматизуємо:** автоматичне формування звітів, контроль проєктів, AI-дашборди.
**Інструменти:** Sheets + Power BI + Make.

### 14. Виробництво / логістика
**Що автоматизуємо:** оновлення запасів, відстеження відвантажень, прогноз попиту.
**Інструменти:** ERP API + Make + ChatGPT для аналітики.

### 15. IT / SaaS-компанії
**Що автоматизуємо:** моніторинг задач, AI-саппорт, генерація технічних документів.
**Інструменти:** Slack API + ChatGPT + Zapier + ClickUp.

### 16. Клініки / медичні центри
**Що автоматизуємо:** нагадування пацієнтам, звітність, AI-аналіз карт.
**Інструменти:** Make + Google Sheets + ChatGPT.

### 17. Туризм / івенти / агентства подій
**Що автоматизуємо:** AI-підбір пропозицій, CRM, боти для бронювань.
**Інструменти:** Make + Google Calendar + ChatGPT + Telegram.

## ВЕЛИКИЙ БІЗНЕС / ENTERPRISE (100+ людей)
Фокус — стратегічна інтеграція AI у системи управління й аналітики.

### 18. Фінтех, банки, страхові компанії
**Що автоматизуємо:** AI-чат-боти, автоматичне звітування, аналіз звернень.
**Інструменти:** n8n + Azure OpenAI + Power BI.

### 19. Корпорації, холдинги, рітейл-мережі
**Що автоматизуємо:** AI-моніторинг KPI, інтеграція CRM + ERP, прогнозування продажів.
**Інструменти:** Data Studio + ChatGPT API + BigQuery.
