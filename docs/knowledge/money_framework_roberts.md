# 📄 Анализ Фреймворка M.O.N.E.Y (Jack Roberts)

## Обзор
Бизнес-модель "AI Agency" с низким барьером входа и высоким удержанием (Retention) через предоставление бесплатной ценности сразу (Free Value First).

---

## 🏗️ Структура M.O.N.E.Y

### 1. M: Mapping (Нишевание)
- **Стратегия**: "Boring Business" (скучные ниши). Выбор сфер с высоким чеком и низким уровнем цифровизации.
- **Критерии ниши**: 
    - Высокая LTV клиента (например, ремонт крыш).
    - Устаревшие сайты у конкурентов.
    - Наличие отзывов на Google Maps (признак живого спроса).

### 2. O: Obtain Leads (Добыча данных)
- **Инструменты**: Outscraper / Apify для Google Maps.
- **Обогащение**: Anymail Finder / Hunter.io для поиска личных email владельцев.
- **Квалификация**: Система должна фильтровать тех, у кого УЖЕ есть отличный сайт, оставляя только тех, кому мы можем помочь.

### 3. N: Nailing the Build (Проектирование)
- **Метод**: "Clone & Customize". Создается один эталонный лендинг для ниши (например, "Клининг").
- **Автоматизация**: n8n меняет в шаблоне: Логотип, Телефон, Название города, Отзывы (спарсенные ранее).
- **Результат**: Ссылка на живой сайт-превью, созданная за 5 минут.

### 4. E: Execute Outreach (Холодный охват)
- **Инструмент**: Instantly.ai (Cold Email).
- **Оффер**: "Я уже сделал для вас сайт, посмотрите по этой ссылке. Если нравится — созвонимся".
- **Эффективность**: Конверсия в ответ в 3-5 раз выше, чем у обычных писем, так как ценность УЖЕ создана.

### 5. Y: Yield (Окупаемость / MRR)
- **Модель**: Подписочная (Subscription). $150–$300/мес за хостинг, правки и поддержку.
- **Апсейл**: Внедрение AI-сотрудников (Chatbots) для обработки заявок — еще $100–300/мес.

---

## 🛠️ Технологический стек для реализации
- **Orchestrator**: n8n.
- **Data**: Google Maps API + Anymail Finder API.
- **Frontend**: WordPress (Elementor/Gutenberg JSON templates) или Framer.
- **Outreach**: Instantly API.
- **Database**: PostgreSQL / Airtable для хранения статусов лидов.
