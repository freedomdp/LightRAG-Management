# Стратегия экспертной ИИ-интеграции и борьбы с галлюцинациями (LightRAG)

### Экспертная интеграция ИИ для бизнеса (LightRAG)

**Задача:** Обогатить базу знаний глубокой технической экспертизой по ИИ-агентам для бизнеса, решить проблемы галлюцинаций, защиты от зависаний (Anti-Freeze) и сквозной идентификации (Omnichannel).

**Решение:**
1.  **Насыщение базы (Deep Ingest):** С помощью `browser_subagent` извлечены полные тексты экспертных статей (Сбер, B1, Selectel, Neuromate, Habr, Reddit). Эти данные индексированы в LightRAG для ответов уровня "Архитектор".
2.  **Паттерн "Watcher-Node":** Для предотвращения "зависаний" ИИ предложена архитектура внешнего контроля времени ответа (таймаут в n8n/Make) с триггером переключения на человека.
3.  **Omnichannel Identity:** Разработана стратегия "Session Stitching" на основе сопоставления внешних ID (TG/WhatsApp) с CRM-ключами через PostgreSQL/HubSpot.
4.  **Борьба с галлюцинациями:** Внедрен 5-шаговый чек-лист верификации от Neuromate и паттерн "Validator-Critic" для мультиагентных систем.
5.  **Coding Agents:** Сравнение Cursor vs Windsurf (режим Cascade) для автоматизации внутренней разработки и поставки ИИ-решений клиентам.

**Затронутые файлы:**
- `docs/knowledge/new_ingest/` (7 новых экспертных статей)
- `scripts/ingest_manager.py` (использован для массовой индексации)
- `scripts/backup_manager.py` (создан полный бэкап данных 2GB)
