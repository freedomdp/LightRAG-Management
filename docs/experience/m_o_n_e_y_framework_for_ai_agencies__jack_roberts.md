# M.O.N.E.Y Framework for AI Agencies (Jack Roberts)

# Анализ Фреймворка M.O.N.E.Y (Jack Roberts)

Модель AI Agency с низким барьером входа и высоким Retention через Free Value First.

## 🏗️ Структура M.O.N.E.Y
1. M: Mapping (Нишевание): Фокус на "Boring Business" (ремонт крыш, клининг) с высоким LTV и низким IT.
2. O: Obtain Leads: Outscraper/Apify (Google Maps) + Anymail Finder/Hunter.io.
3. N: Nailing the Build: "Clone & Customize". n8n адаптирует шаблон под лида (лого, телефон, отзывы).
4. E: Execute Outreach: Instantly.ai (Cold Email). Предложение готового сайта-превью.
5. Y: Yield (Окупаемость): Subscription модель ($150-300/мес) + AI Upsell ($100-300/мес).

## 🛠️ Технологический стек
- Orchestrator: n8n.
- Data: Google Maps API, Anymail Finder.
- Frontend: WordPress/Elementor + Gutenberg JSON templates.
- Outreach: Instantly.ai.
- Database: PostgreSQL/Airtable.
