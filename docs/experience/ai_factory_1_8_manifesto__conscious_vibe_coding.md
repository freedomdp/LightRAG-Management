# AI Factory 1.8 Manifesto (Conscious Vibe Coding)

# Манифест Осознанного Вайб-кодинга (AI Factory 1.8 Standard)

Переход от хаотичного "фикс на фикс" к структурированной разработке.

## 1. Осознанный vs Хаотичный Вайб-кодинг
- Осознанный (Conscious Vibe): Спецификация -> План -> Реализация -> Валидация (HLV).
- Хаотичный (Chaos Vibe): Цикл правок "хочу кнопку", засоренный контекст.

## 2. Spec-Driven Approach
- Проблема: галлюцинации и потеря контекста.
- Решение: Агент не пишет код до подтверждения плана (fixplan.md).

## 3. Механика проксирования Claude (JIT Loading)
- В системный промпт Claude передаются только схемы (schema) скиллов.
- Код скилла/контекст подгружается Just-In-Time в момент Tool Call.
- Позволяет использовать 14+ агентов без роста токенов в базовом промпте.

## 4. Нововведения 1.8
- Git Worktree (feature --parallel): Изоляция агентов для параллельной работы (UI vs Backend).
- Utility-скиллы: Dockerize, CI/Build, Docs (авто-генерация README).
- Support for 14+ Agents: Cursor, Windsurf, Roo Code, Claude Code в едином воркфлоу.

## 5. Философия качества
- Поддержка "быстрого вайба" стоит в 10 раз дороже разработки.
- Обязательная валидация (Layer 3 HLV).
- Wipe контекста при зацикливании и создание нового плана.

Золотой стандарт: Архитектура видна ИИ как на ладони.
