# Динамическое управление ядрами и Anti-Looping в RAG (Ollama + n8n)

Успешно реализована архитектура динамического распределения ресурсов LLM и защиты от зацикливания (Anti-Looping) для проекта Aberhol AI Bot.

### Ключевые решения:
1. **Динамический num_thread**: Настройка n8n для передачи параметров потоков в Ollama API в зависимости от времени суток (Day/Night профили). Это позволяет делить ресурсы сервера с пользователями-людьми (бухгалтер) в рабочее время.
2. **Anti-Looping**: Комбинированный подход. LLM-параметры (stop sequences, repeat penalty) + n8n Filter (фильтрация эха и саморекурсии).
3. **Ryzen 9 7900X Optimization**: Форсирование AVX2 библиотек через переменную окружения `OLLAMA_LLM_LIBRARY=cpu_avx2` и ограничение потоков инференса числом физических ядер (12) для исключения задержек SMT.

### Контекст:
- Стек: n8n, Ollama (Native Windows), LightRAG, Gemma 4.
- ОС: Windows Server (CPU-Only).
