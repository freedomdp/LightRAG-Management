# Корректировка стратегии AI Agency и модели M.O.N.E.Y.

### Ошибка в понимании бизнес-модели
Ранее агент предлагал обобщенные ИИ-решения и недооценивал глубину персонализации. Это «средний уровень» (AI generalist), который не соответствует бизнес-целям данного проекта.

### Истинная модель AI Agency (M.O.N.E.Y.)
1. **Core Product**: Не «советы по ИИ», а высококачественный нишевый сайт.
2. **Free Value First**: Сайт строится заранее. В него интегрируется логотип, контакты и структура, основанная на реальных болях клиентов из отзывов (результат глубокого скрапинга).
3. **Wow-Effect**: Достигается через ИИ-дизайн и SEO-оптимизацию, видимую сразу.
4. **Outreach**: Это не спам, а долгосрочная серия (5-8 писем) с демонстрацией превосходства нашего решения над старым сайтом клиента.
5. **Upsell**: ИИ-агенты, чат-боты и автоматизация предлагаются как надстройка к уже работающему сайту для улучшения бизнес-метрик.

### Техническое следствие для этапа Mapping (M)
Скрапинг должен быть максимально глубоким:
- Сбор всех отзывов для контент-анализа (формирование блоков лендинга).
- Сбор мета-данных старого сайта (для сравнения в письмах).
- Обогащение из Yelp/Facebook для полноты картины.
- Хранение в векторных БД (Qdrant) для поиска похожих успешных кейсов в нише.
